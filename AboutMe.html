# 开源项目

  1. dingtalk (opens new window)，基于网页版钉钉封装的支持 Windows、Linux、Mac 的桌面应用。
  2. screenshots (opens new window), A screenshot plugin for electron.
  3. grender (opens new window)，canvas 二维渲染器，支持图形旋转、缩放和平移，并且完整支持各种鼠标事件。
  4. electron-dev-webpack-plugin (opens new window)，electron 的 webpack 主进程开发自动重启应用的插件，当主进程代码变更之后，就会结束当前主进程，然后重新启动一个新的主进程，方便 electron 开发。
  5. koa-parser (opens new window)，Koa 请求体解析中间件 koa-parser，支持解析 json、form-data、form-urlencoded、text/plain 解析。
  6. markdown365-parser (opens new window)，基于虚拟 dom 的 markdown，vnode 部分参考 vue 的 diff 算法实现，语法解析参考了 marked，项目实用价值很小，虚拟 dom 对性能提示有限。
  7. screenshot-node (opens new window)，开发 node C++截图扩展，目前已支持 Mac，Windows 与 Linux 正在开发。
  8. 垃圾回收小助手微信小程序 (opens new window)，调用阿里云和百度云的图像识别 API 来识别照片，帮助分类。
  9. 涂鸦跳跃微信小游戏 (opens new window)，未使用任何游戏框架。
